Difference between var, let, and const.

Var is a function scoped variable, while let and const are blocked scoped variables. Var variables can be reassigned and can be accessed outside the blocked scope. Where let and const variables can not be reassigned. Const variables are immutable. Var and let variables are mutable.

Difference between regular and arrow function.

The output of regular and arrow functions are similar. However, arrow function is the simplified version of regular function. We cannot use duplicate named parameters inside arrow functions but we can use duplicate named parameters in regular functions. Arrow functions require an arrow symbol which is made of the equal and greater than sign. Arrow function does not require the first bracket for a single parameter where the regular function requires the first bracket for a single parameter.

Difference between map, forEach, filter, and find.

Map, forEach, filter, and find executes the same code for every element of an array. Map and forEach return a new array after executing the code on each element. There is no condition required for map and forEach. However, to use filter and find we have to use conditions. If the elements of the array meet with the condition, then filter returns a new array of the truthy elements. Where find only returns a single value of the matched elements.

Why to use template string?

The use of template string method is known as string interpolation. Template string allows us to embed expression or data dynamically in the form of a string. Also, writing multiline string by using template string is way easier. Template string is written with the backtick symbol.
